# 小红书算法规则（星云 5.0，2025-2026）

## CES 评分公式

```
CES = 关注×8 + 评论×4 + 转发×4 + 点赞×1 + 收藏×1
```

- 高质量评论（≥8 字 + 场景描述）权重是普通评论的 **3 倍**
- 发布后 2 小时内作者回复评论 → 进入推荐瀑布概率 **+30%**
- 65% 的持续流量来自搜索 → **关键词布局决定长尾生命力**

### 核心驱动力

| 指标 | 权重 | 驱动内容设计 |
|------|------|-------------|
| 关注 | 8 | 人格化、系列感、「关注看下一期」 |
| 评论 | 4 | 提问式结尾、争议性观点 |
| 转发 | 4 | 实用干货、「转给需要的人」 |
| 收藏 | 1 | 「以后用得上」的实操内容 |
| 点赞 | 1 | 情绪共鸣、认同感 |

## 推荐机制

### 发布后流量池

1. **种子推荐**（0-30 分钟）：推给粉丝 + 兴趣标签匹配用户
2. **黄金 30 分钟**：前 30 分钟的互动数据占推荐决策的 **60% 权重**
3. **3 秒规则**：视频前 3 秒完播率 < 55% → 直接限流
4. **图文轮播**：滑动 < 3 屏算「无效阅读」→ 限流

### 搜索排名公式

```
排名 = 内容质量(40%) + 用户互动(30%) + 账号权重(20%) + 实时热度(10%)
```

## 发布时间

### 大号（> 1 万粉）
- 19:00-21:00（晚高峰，最高活跃）
- 12:00-13:30（午休）

### 小号（< 1 万粉）——避开高峰，避免被大号抢流量
- 07:00-07:30
- 12:30-13:00
- 17:30-18:30

## 限流触发条件

| 触发 | 影响 |
|------|------|
| 其他平台水印（抖音/微博 logo） | 直接限流 |
| 模糊/低质量图片 | 标记低质量 |
| 禁词（见 banned-words.md） | 限流或删除 |
| 新账号（0-10 篇）频繁修改（≥3 次） | 观察期，流量降至 30-50% |
| 收藏多但无真实评论 | 刷量嫌疑，CES 扣 1.5 分 |
| AI 生成内容未标注且未修改 ≥30% | 限流 50% |
| 视频前 3 秒完播率 < 45% | 直接限流 |

## 图文 vs 视频

| 维度 | 图文 | 视频 |
|------|------|------|
| 总量占比 | 56.67%（4.84 亿篇） | 43.33%（3.7 亿篇） |
| 点赞天花板 | Top 10 平均 22.5 万 | Top 10 平均 66.5 万（**3 倍**） |
| 收藏率 | 高收藏率（**1.8 倍**于视频） | 短期爆发力强 |
| 平台策略 | 保留 30-40% 流量给图文 | 优先但非唯一 |

**结论：视频的爆发力更强，图文的长尾和收藏更好。AI/干货内容两种形式都用。**

## SEO 关键词布局

### 关键词层级

| 类型 | 用户阶段 | 示例 |
|------|----------|------|
| 场景词 | 泛探索 | 冬天穿搭、下班后放松 |
| 品类词 | 对比选择 | AI 工具、效率软件 |
| 品牌词 | 高意向 | Claude、Cursor |
| 长尾词 | 精准搜索 | Claude Code 教程 |

### 关键词放置规则

- **标题前 18 字**必须包含 2 个核心关键词
- 正文每 300 字自然出现 1 次，分布在开头/中间/结尾
- 标签中包含主关键词
- 禁止同段堆砌
